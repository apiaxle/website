In brief

ApiAxle is a proxy that sits on your network*, in front of your API(s) and manages things that you shouldn't have to, like rate limiting, authentication and caching. It's fast, open and easy to configure.

* The on-premise feature of the proxy was key to ApiAxle's model but there's a good chance that cloud hosted versions will be an option now that it's part of the Exicon family.

Open Source

Read the source
Sysadmin concerned about what's happening to your sensitive API data? Show them the source. More eyes on the code means fewer mistakes.
Contribute features
People can donate features, bugs, opinions and documentation to the project. We benefit from them as do our customers. We can ask for copyright transfer and retain the right to re-licence.

Features

  • Caching.
  • Powerful API of its own.
  • Statistics.
  • Keyrings.
  • Authentication - plain key, HMAC signature.
